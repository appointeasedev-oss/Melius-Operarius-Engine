The Pantry API is a free, REST-based JSON storage service for small projects.
It lets you store, retrieve, update, and delete JSON objects in the cloud without setting up your own database.
It’s useful for quick prototypes, small apps, IoT devices, or temporary data storage.

Key Features
Free & Simple: No database setup required.
REST API: Works with any language that can make HTTP requests.
JSON Storage: Store structured data as JSON.
Perishable Data: Data is deleted after a period of inactivity (default ~30 days).
Unlimited Pantries: Each pantry is like a personal database.
How It Works
Create a Pantry

Go to https://getpantry.cloud/
Click Create Pantry → You’ll get a Pantry ID (UUID).
Pantry Structure

Pantry → Your personal storage space.
Basket → A named JSON object inside your pantry.
Base URL

Copy code
https://getpantry.cloud/apiv1/pantry/{PANTRY_ID}
Endpoints
Method	Endpoint	Description
GET	/basket/{basketName}	Retrieve JSON from a basket
POST	/basket/{basketName}	Create or overwrite JSON in a basket
PUT	/basket/{basketName}	Update JSON in a basket
DELETE	/basket/{basketName}	Delete a basket
Example in Python
Python

Copy code
import requests
import json

PANTRY_ID = "your-pantry-id-here"
BASKET_NAME = "myData"
BASE_URL = f"https://getpantry.cloud/apiv1/pantry/{PANTRY_ID}/basket/{BASKET_NAME}"

# Data to store
data = {
    "name": "John Doe",
    "age": 30,
    "skills": ["Python", "API", "Cloud"]
}

try:
    # Store data (POST)
    response = requests.post(BASE_URL, json=data)
    response.raise_for_status()
    print("Data stored successfully:", response.json())

    # Retrieve data (GET)
    response = requests.get(BASE_URL)
    response.raise_for_status()
    print("Retrieved data:", response.json())

except requests.exceptions.RequestException as e:
    print("Error:", e)
Example in JavaScript (Node.js)
Javascript

Copy code
const axios = require('axios');

const PANTRY_ID = "your-pantry-id-here";
const BASKET_NAME = "myData";
const BASE_URL = `https://getpantry.cloud/apiv1/pantry/${PANTRY_ID}/basket/${BASKET_NAME}`;

(async () => {
  try {
    // Store data
    let res = await axios.post(BASE_URL, {
      name: "John Doe",
      age: 30,
      skills: ["JavaScript", "API", "Cloud"]
    });
    console.log("Data stored:", res.data);

    // Retrieve data
    res = await axios.get(BASE_URL);
    console.log("Retrieved data:", res.data);

  } catch (err) {
    console.error("Error:", err.message);
  }
})();
